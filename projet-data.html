Gestion de Données (Base de Données)

Conception et exploitation de Bases de Données relationnelles

Contexte du Projet

Dans le cadre des modules de Base de Données (SAE BD), on nous a donnée plusieurs fichiers excel non organnisé qu'il fallait donc introduire dans une base de données grâce a dbeaver, la tache était donc de créer une base de données le plus optimisé possible pour organnisé ses données sans répétitions et sans en perdre.

  • Cadre : SAE Base de Données
  • Outils : Looping, Oracle
  • Langages : SQL

Méthodes et Résultats

Le projet s'est déroulé en plusieurs phases :

  1. Analyse des besoins et dictionnaire des données.
  2. Modélisation conceptuelle (MCD) avec le logiciel Looping.
  3. Traduction en Modèle Logique de Données (MLD).
  4. Implémentation en SQL (création de tables, contraintes).
  5. Phase 2 : utilisation de la BD et création de vues pour exploité les données.

Compétences Travaillées

  • Modélisation : Conception de schémas relationnels normalisés.
  • SQL Avancé : Jointures complexes, héritage.
  • Optimisation : trouver la meilleur configuration possible.

Contribution Personnelle

J'ai réalisé la conception du MCD sur Looping et écrit tout le scripts de création de tables. J'ai également travaillé sur des requêtes d'analyse de données pour extraire des statistiques pertinentes pour la deuxième partie du projet.
