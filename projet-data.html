Gestion de Données (Base de Données)

Conception et exploitation de Bases de Données relationnelles

Contexte du Projet

Dans le cadre des modules de Base de Données (SAE BD), nous avons appris à gérer le cycle de vie complet de la donnée, de la modélisation conceptuelle à l'exploitation via des requêtes complexes.

  • Cadre : SAE Base de Données
  • Outils : Looping, PostgreSQL
  • Langages : SQL, PL/SQL

Méthodes et Résultats

Le projet s'est déroulé en plusieurs phases :

  1. Analyse des besoins et dictionnaire des données.
  2. Modélisation conceptuelle (MCD) avec le logiciel Looping.
  3. Traduction en Modèle Logique de Données (MLD).
  4. Implémentation en SQL (création de tables, contraintes).
  5. Développement de procédures stockées et triggers en PL/SQL pour l'automatisation.

Compétences Travaillées

  • Modélisation : Conception de schémas relationnels normalisés.
  • SQL Avancé : Jointures complexes, agrégats, sous-requêtes.
  • Administration : Gestion des droits et des utilisateurs.
  • Optimisation : Analyse de l'occupation mémoire.

Contribution Personnelle

J'ai réalisé la conception du MCD sur Looping et écrit une grande partie des scripts de création de tables. J'ai également travaillé sur des requêtes d'analyse de données pour extraire des statistiques pertinentes.
